PROJECT TITLE: AI-DRIVEN MEDICAL RECORD SCANNER

DESCRIPTION:
AI driven web app which scans through PDF's of long patient medical history records and creates a highlight of significant diagnosis as well as suggesting new diagnosis.

TECH STACK:

*Frontend: HTML/PHP
*Backend: PHP and JavaScript
*Database: Supabase
*Other tools: XAMPP app & Composer app

FEATURES: 
*A simple login system
*Uploading of medical records
*AI based diagnosis detection
*Past diagnoses detection
*Retrievable patient data

SYSTEM REQUIREMENTS:
*Use XAMPP v3.3.0
